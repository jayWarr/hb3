User Parameters

These are a number of switches and settings that can be changed to customise the behaviour of the application to your requirements


Important many of the user parameters do not have to be changed, all are initially set up with sensible defaults. Some of them cannot be changed by you and are controlled by the application.


However several of them work in concert with actions and or housekeeping functions and may need changing to achieve the effect you desire.


E.g. you can change the default alert window for competitions closing, from the default 10 days, to a longer or shorter period to fit in with how you respond to closing dates.
